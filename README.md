# convertToTagFormByBlockly
ブロックを組み合わせてフォームをプログラムし、Webサイトがコードを自動生成します。そのコードをコピペするだけで、簡単にフォームを表示できます。
マイクラの[Tag Form Engine]アドオンのリンクはコチラ↓↓  
https://minecraft-mcworld.com/203474/

# Webサイトの説明
## カテゴリー
Webサイトには、「総合」「アクションフォーム」「モーダルフォーム」「その他」の4つのカテゴリーがあります。

総合...　タイトルやコマンドを設定する等、それぞれのフォームで使えるブロックを集めているカテゴリーです。

アクションフォーム...　ボタンを選択する「アクションフォーム」でしか使用できないブロックを集めているカテゴリーです。

モーダルフォーム...　スライダーやスイッチ、テキストボックス、ドロップダウンを追加する「モーダルフォーム」でしか使用できないブロックを集めているカテゴリーです。

その他...　条件や、スコアボード取得など、フォームを表示するときによく使う「変数」など、その他の物を集めたフォームです。
